## Установка

1. установить [docker](https://www.docker.com/) и docker-compose

2. установить [node](https://nodejs.org/uk/)

3. выполнить консольные команды:
``` 
git clone https://github.com/icekhab/teamu-front.git

# установить пакеты
cd ./src/ && npm i && ../

# поднять контейнеры
docker-compose up -d
```
